ChartAnimation for FlexChart

Important! This feature requires the wijmo.chart.animation extension module.

This view shows ChartAnimation for FlexChart.

You can use different animation modes for FlexChart by setting the animationMode property of the ChartAnimation.

The ChartAnimation class has a duration property that allows you to set the number of periods for animation.
