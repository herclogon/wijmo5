Angular 2 component for the WjFlexChart DataPoint objects.

The wj-flex-chart-data-point component must be contained in a WjFlexChart component.

Use the wj-flex-chart-data-point component to add DataPoint objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartDataPoint component is derived from the DataPoint class and inherits all its properties, events and methods.
