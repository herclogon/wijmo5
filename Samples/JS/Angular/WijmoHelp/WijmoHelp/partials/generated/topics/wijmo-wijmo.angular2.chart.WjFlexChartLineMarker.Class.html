Angular 2 component for the WjFlexChart LineMarker control.

The wj-flex-line-marker component must be contained in a WjFlexChart component.

Use the wj-flex-line-marker component to add LineMarker objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartLineMarker component is derived from the LineMarker class and inherits all its properties, events and methods.

Properties

Name Type Description
alignment LineMarkerAlignment

Gets or sets the alignment of the LineMarker content.

By default, the LineMarker shows to the right, at the bottom of the target point. Use '|' to combine alignment values.

// set the alignment to the left.
marker.alignment = wijmo.chart.LineMarkerAlignment.Left;
// set the alignment to the left top.
marker.alignment = wijmo.chart.LineMarkerAlignment.Left | wijmo.chart.LineMarkerAlignment.Top;
