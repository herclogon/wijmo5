Angular 2 component for the WjFlexChart PlotArea objects.

The wj-flex-chart-plot-area component must be contained in a WjFlexChart component.

Use the wj-flex-chart-plot-area component to add PlotArea objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartPlotArea component is derived from the PlotArea class and inherits all its properties, events and methods.
