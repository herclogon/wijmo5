Angular 2 component for the WjFlexChart Circle annotation objects.

The wj-flex-chart-annotation-circle component must be contained in a WjFlexChartAnnotationLayer component.

Use the wj-flex-chart-annotation-circle component to add Circle annotation objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartAnnotationCircle component is derived from the Circle class and inherits all its properties, events and methods.
