Angular 2 component for the WjFlexChart Range object.

The wj-range component must be contained in a WjLinearGauge, WjRadialGauge or WjBulletGraph component.

Use the wj-range component to add Range objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjRange component is derived from the Range class and inherits all its properties, events and methods.
