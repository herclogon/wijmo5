IRenderEngine Class

Module
wijmo.chart
File
wijmo.chart.js

Represents a rendering engine that performs the basic drawing routines.

Properties

Name Type Description
F element Element Gets the rendered element.
F fill string Gets or sets the color used to fill the element.
F fontFamily string Gets or sets the font family for the text output.
F fontSize string Gets or sets the font size for the text output.
F stroke string Gets or sets the color used to outline the element.
F strokeWidth number Gets or sets the thickness of the outline.
F textFill string Gets or sets the text color.

Methods

Name Parameters Description
beginRender​() Clear the viewport and start the rendering cycle.
endRender​() Finish the rendering cycle.

w (number): Viewport width.

 Set the size of the viewport.
