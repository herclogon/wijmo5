AllowResizing Enum

Module
wijmo.grid
File
wijmo.grid.js

Specifies constants that define the row/column sizing behavior.

Values

Name Description
Both The user may resize rows and columns by dragging the edge of the headers.
BothAllCells The user may resize rows and columns by dragging the edge of any cell.
Columns The user may resize columns by dragging the edge of the column headers.
ColumnsAllCells The user may resize columns by dragging the edge of any cell.
None The user may not resize rows or columns.
Rows The user may resize rows by dragging the edge of the row headers.
RowsAllCells The user may resize rows by dragging the edge of any cell.
