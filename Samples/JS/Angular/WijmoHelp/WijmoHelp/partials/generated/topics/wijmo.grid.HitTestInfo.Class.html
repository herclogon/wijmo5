HitTestInfo Class

Module
wijmo.grid
File
wijmo.grid.js

Contains information about the part of a FlexGrid control that exists at a specified page coordinate.

Constructor

Name Parameters Description
constructor​(g, pt): HitTestInfo

g (any): The FlexGrid control or GridPanel to investigate.
pt (any): The Point object in page coordinates to investigate.

 Initializes a new instance of the HitTestInfo class.

Properties

Name Type Description
cellType CellType Gets the cell type at the specified position.
col number Gets the column index of the cell at the specified position.
edgeBottom boolean Gets a value that indicates whether the mouse is near the bottom edge of the cell.
edgeLeft boolean Gets a value that indicates whether the mouse is near the left edge of the cell.
edgeRight boolean Gets a value that indicates whether the mouse is near the right edge of the cell.
edgeTop boolean Gets a value that indicates whether the mouse is near the top edge of the cell.
panel GridPanel Gets the grid panel at the specified position.
point Point Gets the point in control coordinates that the HitTestInfo refers to.
range CellRange Gets the cell range at the specified position.
row number Gets the row index of the cell at the specified position.
