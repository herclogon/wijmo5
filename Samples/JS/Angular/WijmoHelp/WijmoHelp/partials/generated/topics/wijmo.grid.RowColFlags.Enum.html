RowColFlags Enum

Module
wijmo.grid
File
wijmo.grid.js

Specifies flags that represent the state of a grid row or column.

Values

Name Description
AllowDragging The row or column can be dragged to a new position with the mouse.
AllowMerging The row or column can contain merged cells.
AllowResizing The row or column can be resized.
AllowSorting The column can be sorted by clicking its header with the mouse.
AutoGenerated The column was generated automatically.
Collapsed The group row is collapsed.
ColumnDefault Default settings for new columns.
HtmlContent Cells in this row or column contain HTML text.
ParentCollapsed The row has a parent group that is collapsed.
ReadOnly The row or column is read-only (cannot be edited).
RowDefault Default settings for new rows.
Selected The row or column is selected.
Visible The row or column is visible.
WordWrap Cells in this row or column may contain wrapped text.
