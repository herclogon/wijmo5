FlexSheet

Defines the FlexSheet control.

The FlexSheet control extends the FlexGrid control to provide Excel-like features such as a calculation engine, multiple sheets, undo/redo, and XLSX import/export.
FlexSheetPanel Defines the extension of the GridPanel class, which is used by FlexSheet where the base FlexGrid class uses GridPanel. For example, the cells property returns an instance of this class.
HeaderRow Represents a row used to display column header information for a bound sheet.
Sheet Represents a sheet within the FlexSheet control.
SheetCollection Defines the collection of the Sheet objects.
SortManager Maintains sorting of the selected Sheet of the FlexSheet.
UndoStack Controls undo and redo operations in the FlexSheet.
UnknownFunctionEventArgs Provides arguments for unknown function events.
