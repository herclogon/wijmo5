FlexGridXlsxConverter Class

Module
wijmo.grid.xlsx
File
wijmo.grid.xlsx.js

This class provides static load and save methods for loading and saving of the FlexGrid control from/to Excel xlsx files.

Methods

Name Parameters Description
S load​(grid, workbook, options?)

grid (FlexGrid): FlexGrid that will load the workBook object.
workbook (any): An Workbook instance or a Blob instance or a base 64 stirng or an ArrayBuffer containing xlsx file content.
options? (IFlexGridXlsxOptions): Options to use when load the workBook object to FlexGrid instance (including whether to save row and column headers, sheet name, etc)

Loads an Workbook instance or a Blob object containing xlsx file content to FlexGrid instance.

For example: 

// This sample opens an xlsx file chosen via Open File
// dialog and fills FlexGrid with the content of the first
// sheet.
 
// HTML
<input type="file" 
    id="importFile" 
    accept="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet" 
/>
<div id="flexHost"></>
 
// JavaScript
var flexGrid = new wijmo.grid.FlexGrid("#flexHost"),
    importFile = document.getElementById('importFile');
 
importFile.addEventListener('change', function () {
    loadWorkbook();
});
 
function loadWorkbook() {
    var reader,
        file = importFile.files[0];
    if (file) {
        reader = new FileReader();
        reader.onload = function (e) {
            wijmo.grid.xlsx.FlexGridXlsxConverter.load(flexGrid, reader.result,
                { includeColumnHeaders: true });
        };
        reader.readAsArrayBuffer(file);
    }
}
S save​(grid, options?, fileName?): Workbook

grid (FlexGrid): FlexGrid that will be saved.
options? (IFlexGridXlsxOptions): Options to use when saving the grid (including whether to save row and column headers, sheet name, etc)
fileName? (): Name of the file that will be generated.

Save the FlexGrid instance to Workbook instance.

For example: 

// This sample exports FlexGrid content to an xlsx 
// click.
 
// HTML
<button 
    onclick="saveXlsx('FlexGrid.xlsx')">
    Save
</button>
 
// JavaScript
function saveXlsx(fileName) {
    // Save the flexGrid to xlsx file.
    wijmo.grid.xlsx.FlexGridXlsxConverter.save(flexGrid,
            { includeColumnHeaders: true }, fileName);
}



Returns A Workbook object that can be used to customize the workbook before saving it (with the Workbook.save method).
