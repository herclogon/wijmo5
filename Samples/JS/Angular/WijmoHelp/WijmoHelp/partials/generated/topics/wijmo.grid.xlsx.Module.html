Module wijmo.grid.xlsx

File
wijmo.grid.xlsx.js

Extension that defines the FlexGridXlsxConverter class that provides client-side Excel xlsx file save/load capabilities for the FlexGrid control.

Classes

Name Description
FlexGridXlsxConverter This class provides static load and save methods for loading and saving of the FlexGrid control from/to Excel xlsx files.

Interfaces

Name Description
IExtendedSheetInfo Defines additional worksheet properties that can be accesses via the dynamic wj_sheetInfo property of the FlexGrid instance.
IFlexGridXlsxOptions FlexGrid Xlsx conversion options

Properties

Name Type Description
F activeWorksheet number Index of the active sheet in the xlsx file.
F dataType DataType Data Type
F format string Format
F hAlign string Horizontal alignment
F includeCellStyles boolean Indicates whether cells styling should be included in the generated xlsx file.
F includeColumnHeader boolean Indicates whether to treat the first xlsx file row as containing column header names.
F includeColumnHeaders boolean Indicates whether to include column headers as first rows in the generated xlsx file.
F includeRowHeaders boolean Indicates whether to include row headers as first columns in the generated xlsx file.
