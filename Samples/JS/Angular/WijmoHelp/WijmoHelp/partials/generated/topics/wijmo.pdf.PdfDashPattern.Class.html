(default value: null)

 Initializes a new instance of the PdfDashPattern class.

Properties

Name Type Description
dash number Gets or sets the length of alternating dashes, in points. The default value is null which indicates no dash pattern, but a solid line.
gap number Gets or sets the length of alternating gaps, in points. The default value is equal to dash which indicates that dashes and gaps will have the same length.
phase number Gets or sets the distance in the dash pattern to start the dash at, in points. The default value is 0.
