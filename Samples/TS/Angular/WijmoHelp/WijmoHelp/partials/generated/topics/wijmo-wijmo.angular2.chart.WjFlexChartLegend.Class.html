Angular 2 component for the WjFlexChart Legend control.

The wj-flex-chart-legend component must be contained in a WjFlexChart component.

Use the wj-flex-chart-legend component to add Legend objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartLegend component is derived from the Legend class and inherits all its properties, events and methods.
