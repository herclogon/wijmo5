Angular 2 component for the WjFlexChart RangeSelector object.

The wj-flex-chart-range-selector component must be contained in a WjFlexChart component.

Use the wj-flex-chart-range-selector component to add RangeSelector objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartRangeSelector component is derived from the RangeSelector class and inherits all its properties, events and methods.
