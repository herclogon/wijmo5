Defines methods and properties that extend ICollectionView to provide paging capabilities.

Properties

Name Type Description
F canChangePage boolean ` * Gets a value that indicates whether the pageIndex value can change.
F isPageChanging boolean ` * Gets a value that indicates whether the index is changing.
F itemCount number

` * Gets the number of items in the view taking paging into account.

To get the total number of items, use the totalItemCount property.

Notice that this is different from the .NET IPagedCollectionView, where itemCount and totalItemCount both return the count before paging is applied.
F pageChanged Event Occurs after the page index changes.
F pageChanging Event Occurs before the page index changes.
F pageIndex number ` * Gets the zero-based index of the current page.
F pageSize number ` * Gets or sets the number of items to display on a page.
F totalItemCount number

` * Gets the total number of items in the view before paging is applied.

To get the number of items in the current view not taking paging into account, use the itemCount property.

Notice that this is different from the .NET IPagedCollectionView, where itemCount and totalItemCount both return the count before paging is applied.
