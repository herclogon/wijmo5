PdfGradientStop Class

Module
wijmo.pdf
File
wijmo.pdf.js

Represents an object which determines a transition point of a gradient.

Constructor

Name Parameters Description
constructor​(offset?, color?, opacity?): PdfGradientStop

offset? (number): The location of the gradient stop on the gradient axis.
color? (any): The color of the gradient stop. A Color object or any string acceptable by the fromString method.
opacity? (number): The opacity of the gradient stop.

 Initializes a new instance of the PdfGradientStop class.

Properties

Name Type Description
color Color Gets or sets the color of the gradient stop. The default color is black.
offset number Gets or sets the location of the gradient stop on gradient axis of the brush. The value must be in range [0, 1], where 0 indicates that the gradient stop is placed at the beginning of the gradient axis, while 1 indicates that the gradient stop is placed at the end of the gradient axis. The default value is 0.
opacity number Gets or sets the opacity of the gradient stop. The value must be in range [0, 1], where 0 indicates that the gradient stop is completely transparent, while 1 indicates that the gradient stop is completely opaque. The default value is 1.
